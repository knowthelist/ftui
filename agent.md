# Agent Instructions

## Language & Compatibility
*   **Runtime Environment:** Ensure all code is compatible with **Chrome v70**.
*   **JavaScript Constraints:** Use only ECMAScript features supported as of late 2018. 
    *   **Allowed:** Classes, Arrow Functions, Template Literals, Destructuring, and Promises.
    *   **Avoid:** Optional Chaining (`?.`), Nullish Coalescing (`??`), and Private Class Fields (`#`), as these require Chrome 80+.
